# CardsTablesMonitor
### marks
* green -  active table players more then 1
* grey -  empty table without players
* blink red - table with warning

### How to run? 
npm install 
npm start

#### used 
a webpack-react-redux (fixed) boilerplate https://github.com/jpsierens/webpack-react-redux
